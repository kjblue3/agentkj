# Demo

1. Mention the agent in a channel with an investigation question.
2. Show that acknowledgement, progress, sources, and the answer stay in one public thread.
3. Privately ask to connect a fictional service and show that setup/authorization information is visible only to the requesting member.
4. As a workspace administrator, open the one-time setup page and review its callback URL and declared hosts.
5. Authorize two fictional member accounts, then ask a question that requires corroboration from both connections.
6. Demonstrate that a missing scope pauses the job, privately requests reauthorization, and resumes after callback.
7. Simulate all LLM keys returning 429 and show the public retry-time warning followed by automatic resumption.
